Eg.1 - HTML5 placeholder attribute

Eg.2 - HTML value attribute

$("#tbox2").placeholder({placeholderProperty: "value"});

Eg.3 - Passing default value as function parameter in script

$("#tbox3").placeholder({defaultValue: "Say something cool.."});
